# QuickLib

QucikLib client application built with React Native and Expo. 

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Buil apk file

   ```bash
           run: eas build --platform android --profile preview --local --output app-release.apk
   ```

3. Install to device

   ```bash
   adb install app-release.apk
   ```
